# 🎤 Speech Translator Pro

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Chrome Support](https://img.shields.io/badge/Chrome-Supported-brightgreen.svg)](https://developer.chrome.com/)
[![Responsive](https://img.shields.io/badge/Responsive-Yes-blue.svg)](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Flexible_Box_Layout)

Advanced real-time **Speech-to-Text**, **Translation**, and **Text-to-Speech** web application built with modern Web Speech API. Features glassmorphism UI, live volume visualization, multi-language support, and session management.

![Screenshot](https://via.placeholder.com/1200x600/667eea/ffffff?text=Speech+Translator+Pro)

## ✨ Features

- **🔴 Real-time Speech Recognition** - Instant transcription
- **🌐 Multi-Language Support** - 9+ languages (English, Hindi, Spanish, French, etc.)
- **⚡ Live Translation** - Auto-detect & translate speech
- **🔊 Text-to-Speech** - Natural voice synthesis
- **📱 Fully Responsive** - Mobile-first design
- **🎨 Advanced UI/UX** - Glassmorphism, animations, dark mode ready
- **📊 Volume Meter** - Real-time audio visualization
- **⏱️ Session Timer** - Track usage duration
- **💾 Export Sessions** - Save as JSON
- **⚙️ Custom Settings** - Voice speed, pitch, continuous mode
